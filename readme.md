# 🎮 Tic-Tac-Toe Game

A simple terminal-based Tic-Tac-Toe game built with Python. Play against a simple AI agent and try to win!

## 🚀 How to Run

To start the game, run the following command in your terminal:

```bash
python tic_tac_toe.py
```

*Note: If `python` doesn't work, try `py tic_tac_toe.py`.*

## 🕹️ How to Play

1. The game is played on a 3x3 grid.
2. You are **X**, and the computer is **O**.
3. When it's your turn, enter a number between **1-9** to choose your spot. The grid positions are mapped as follows:

   ```text
   1 | 2 | 3
   ---------
   4 | 5 | 6
   ---------
   7 | 8 | 9
   ```

4. The first player to get 3 of their marks in a row (up, down, across, or diagonally) is the winner.
5. If all 9 squares are full and no player has 3 marks in a row, the game is a tie.

## 🛠️ Features

- Simple terminal interface.
- Random-move AI opponent.
- Automatic win and tie detection.
